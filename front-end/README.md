# 🏢 Cowork React

Aplicación web de gestión de espacios cowork desarrollada con React.  
El proyecto implementa un sistema de autenticación de usuarios que incluye login, registro y recuperación de contraseña, utilizando almacenamiento local.

---

## 🚀 Funcionalidades

- 🔐 Login de usuario
- 📝 Registro de nuevos usuarios
- 📩 Recuperación de contraseña (simulada)
- 💾 Persistencia de datos con LocalStorage
- 🎨 Diseño minimalista con estilo moderno (temática cowork)
- 🔒 Buenas prácticas de seguridad en frontend (mensajes genéricos, validaciones)

---

## 🛠 Tecnologías utilizadas

- React
- JavaScript (ES6+)
- React Router DOM
- CSS
